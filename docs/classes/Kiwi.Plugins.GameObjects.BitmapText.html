Kiwi.Plugins.GameObjects.BitmapText Class

Extends Entity
Defined in: src/bitmap-text-1.2.0.js:44
Module: Plugins
Parent Module: Kiwi

GameObject to display bitmap text. Works in a very similar fashion to the default TextField GameObject included in the core of KiwiJS, except this one uses a TextureAtlas instead of a font.

You can also set the maximum width of the Text to span multiple lines.
